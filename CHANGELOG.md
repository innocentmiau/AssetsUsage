# Changelog

All notable changes to this package are documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-07-09

### Added

- Initial release.
- Editor window (**Tools > Assets Usage**) that reports per-asset reference counts.
- GUID reference-counting across scenes, prefabs, materials, ScriptableObjects and `.meta` files.
- Folder and asset-type filters.
- Sorting by least or most used.
- CSV export.
- Background-threaded scanning to keep the editor responsive.
